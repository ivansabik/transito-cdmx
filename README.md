App para consultar infracciones y adeudos de tenencia de coches con números de placa de la CDMX.

Primero necesitas hacer un deploy de la API:

```
cd api
npm install
npm start
```

Después de la aplicación cliente:

```
cd tranzita
bower install
php -S localhost:8000
```
